How to add / change tracks
--------------------------
Drop more CC0 / public-domain MP3s here and edit the PLAYLIST array at the top
of the <script> in tapvid-fm.html. Keep files LOCAL — the EQ bars + Tappy's
motion are driven by a Web Audio analyser, which only reads same-origin audio.

More CC0 sources: musopen.org (filter Public Domain / CC0), archive.org
(check each item's rights statement).
